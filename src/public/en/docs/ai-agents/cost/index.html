The Cost Agent helps you understand and optimize your cloud spending across all providers.

Capabilities

Category Operations
Cost Analysis Break down spending by service, team, project
Waste Detection Find unused resources, oversized instances
Right-Sizing Recommend optimal instance sizes
Reserved Capacity RI/Savings Plan recommendations
Budget Management Set budgets, alert on thresholds
Forecasting Predict future spending

Deployment

c2g agent deploy cost \
  --name cost-optimizer \
  --providers aws,azure,gcp

Configuration

name: cost-optimizer
type: cost
providers:
  - name: aws
    cost_explorer: true
    billing_account: true
  - name: azure
    cost_management: true
  - name: gcp
    billing_export: true
settings:
  # Currency for reports
  currency: USD
  # Alert when spending exceeds forecast by this %
  forecast_alert_threshold: 20
  # Consider resources idle after this period
  idle_threshold: 7d
notifications:
  slack: "#cost-alerts"
  email: ["finance@company.com"]
