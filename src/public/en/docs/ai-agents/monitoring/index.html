The Monitoring Agent provides observability and incident response capabilities for your infrastructure.

Capabilities

Category Operations
Observability Query metrics, logs, and traces
Alerting Set up alerts and notifications
Incident Response Automated diagnostics and remediation
SLO Tracking Monitor service level objectives
On-Call Support Assist with incident investigation
Reporting Generate reliability reports

Deployment

c2g agent deploy monitoring \
  --name sre-assistant \
  --providers aws,datadog

Configuration

name: sre-assistant
type: monitoring
providers:
  - name: aws
    services: [cloudwatch, xray]
  - name: datadog
    api_key: ${DATADOG_API_KEY}
  - name: pagerduty
    api_key: ${PAGERDUTY_API_KEY}
settings:
  # Default time range for queries
  default_lookback: 1h
  # Auto-investigate on alert
  auto_investigate: true
notifications:
  slack: "#sre"
  pagerduty: platform-team
